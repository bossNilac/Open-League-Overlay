# Screenshots

Place sanitized release-build screenshots in this folder.

Recommended files:

- `gui-dashboard.png`
- `overlay-team-mode.png`
- `overlay-lane-mode.png`
- `gui-scoreboard.png`
- `tui-scoreboard.png`
- `match-history.png`
- `match-report.png`
- `champ-select-overlay.png`
- `champ-select-gui.png`

Capture guidelines:

- Do not include Riot account details.
- Blur or replace player names if needed.
- Avoid showing private chat, friend list, desktop notifications, or personal folders.
- Prefer PNG format.
- Prefer 1280px or 1920px width.
- Use screenshots from the release build, not temporary debug windows.
- Keep the overlay readable while still showing that it is lightweight.

Do not commit raw private match data or uncensored screenshots.
